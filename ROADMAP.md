## Phase 2 — 真实数据接入 🔨 下一步

| 任务 | 说明 |
|------|------|
| 集成 akshare Python 库 | 免费获取A股实时/历史行情 |
| 编写数据采集脚本 | `scripts/fetch_data.py` — 日线/周线/财务数据 |
| GitHub Actions 定时任务 | 每周一开盘前自动更新数据 |
| 历史K线数据存储 | 输出为 `data/` 目录下的 JSON 文件 |
| 财务指标自动拉取 | PE/PB/ROE/营收增速/净利增速/毛利率 |
| 数据 API 抽象层 | 前端统一从 JSON 读取，支持本地与远程 |

---

## Phase 3 — 量化选股引擎 📋 规划中

| 任务 | 说明 |
|------|------|
| 均线交叉策略 | 自动检测金叉/死叉信号 |
| MACD 指标 | 计算 DIF/DEA/MACD 柱状图 |
| RSI 指标 | 超买超卖判断 |
| KDJ 指标 | 短期趋势信号 |
| 基本面筛选规则 | ROE>15%, PE<30, 营收增速>10% 等可配置 |
| 多因子综合评分 | 技术面 40% + 基本面 40% + 资金面 20% |
| 自动排名输出 TOP 5 | 每周自动筛选并生成推荐列表 |
| 回测框架 | 验证策略历史表现 |

---

## Phase 4 — 新闻与舆情分析 📋 规划中

| 任务 | 说明 |
|------|------|
| 财经新闻采集 | 东方财富/同花顺/新浪财经 RSS/API |
| NLP 情感分析 | 利好/利空/中性自动分类 |
| 新闻事件标注 | 在K线图上标注重大事件 |
| 舆情热度指数 | 综合多源数据计算关注度 |
| 实时新闻流 | 页面内嵌新闻滚动条 |

## Phase 5 — 高级可视化与交互 📋 规划中

## Phase 6 — 智能化与自动化 📋 规划中

| 任务 | 说明 |
|------|------|
| AI 周报生成 | 利用 LLM 自动生成每周选股分析报告 |

Phase 6: + LLM API + ML (scikit-learn/pytorch)
